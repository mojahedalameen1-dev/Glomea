# KidneyTrack Monorepo

## Structure
- `backend/`: Node.js Express API
- `mobile/`: Flutter Mobile Application

## Getting Started
### Prerequisites
- Node.js 20+
- Flutter 3.x
- Docker & Docker Compose

### Backend
1. `cd backend`
2. `npm install`
3. `cp .env.example .env`
4. `npm run dev`

### Mobile
1. `cd mobile`
2. `flutter pub get`
3. `flutter run`
